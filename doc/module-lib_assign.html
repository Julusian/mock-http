(inner) assign(target) → {Object}

Assigns properties of obj1 to objn to target. target needs to an object. If a property already exists in target it will NOT get overwritten!

Example:

var target = {three: 3, one: 1};
assign(target, {one: 0}, {two: 2});
//> target = {one: 1, two: 2, three: 3}
Parameters:

target Object

target obj where props get appended
